# 宝可梦VGC状态异常指南

## I. 状态异常概述

### A. 什么是状态异常？

状态异常是宝可梦对战中施加给对手的负面效果，分为**非挥发性**和**挥发性**两大类。这些效果通过招式、特性、道具或场地施加，能显著改变对战节奏。在VGC（电子游戏锦标赛）双打环境中，状态异常是队伍构建和临场决策的核心工具，尤其在当前规则G/H（2024-2025）的短战节奏下，被用于速度控制、输出削弱和威胁移除。

非挥发性状态异常是“典型”状态，例如烧伤或睡眠，在宝可梦**切换出场后仍然持续**，整个对战期间都有效，且每只宝可梦同时只能拥有一种。挥发性状态异常种类多样（如挑衅、寄生种子），在宝可梦切换出场后即被移除，并且可以多重叠加。

### B. VGC规则下的状态异常基础

VGC采用“6选4”的队伍预览机制，并设有20分钟总时限和每回合45秒的操作时间，强调高效控场。状态异常的施加会受到**神秘守护**、**薄雾场地**、**叶子防守**等防护效果的影响而无法生效（除了特定的例外情况，如睡觉招式）。第九世代（《朱／紫》）的机制较为稳定：睡眠固定为1-3回合，麻痹仅使速度减半（而非旧版的降低至25%），烧伤每回合损失1/16的最大HP。

**免疫与特性影响：**
- 火属性免疫烧伤；电属性免疫麻痹；毒/钢属性免疫中毒。
- 毅力特性（处于烧伤/麻痹状态时攻击提升1.5倍，并且硬撑招式威力翻倍）；魔法防守特性（不会受到状态异常带来的HP损失）。

## II. 非挥发性状态异常详解

这些状态异常持久性强，适合在VGC中用于针对核心输出手进行长期压制。

### A. 烧伤

**效果：** 每回合损失1/16最大HP，物理攻击减半（拥有毅力特性时为例外，并且硬撑招式基础威力变为140）。
**免疫：** 火属性宝可梦、水幕/水泡特性。
**施加方式：** 鬼火（100%命中）、热水/神圣之火招式的附加效果、火焰宝珠。
**移除方式：** 灼伤药、莓榴果、治愈铃声/芳香治疗。
**VGC应用：** 炽焰咆哮虎首发使用鬼火，针对多龙巴鲁托/铁掌力士等物理威胁进行削弱。

### B. 中毒 & 剧毒

**效果：** 中毒每回合损失1/8最大HP；剧毒第一回合损失1/16最大HP，之后每回合递增1/16。
**免疫：** 毒/钢属性宝可梦、免疫特性。
**施加方式：** 剧毒招式、污泥炸弹的附加效果、毒菱（1层导致中毒，2层导致剧毒）。
**移除方式：** 解毒药、桃桃果、治愈铃声。
**VGC应用：** 在短局对战中较少使用（需要多回合积累伤害），但毒菱战术的赛富豪常见于耐久型队伍。

### C. 麻痹

**效果：** 速度减半，25%几率完全麻痹（当回合无法行动）。
**免疫：** 电属性宝可梦、柔软特性。
**施加方式：** 电磁波（100%命中）、打雷招式的附加效果。
**移除方式：** 麻痹药、樱子果。
**VGC应用：** 核心的速度控制手段，用于降低多龙巴鲁托/振翼发等高速宝可梦的速度，确保己方速度线优势。

### D. 睡眠

**效果：** 1-3回合内无法行动（回合数随机，切换出场不消耗睡眠回合）。
**免疫：** 不眠/活力特性、电气场地。
**施加方式：** 蘑菇孢子（100%命中，通常限于败露球菇）、催眠粉、催眠术、哈欠（延时生效）。
**移除方式：** 解眠药、零余果。
**VGC应用：** 强力移除威胁，例如败露球菇使用蘑菇孢子来拖延对手的战术强化。

### E. 冰冻

**效果：** 无法行动，每回合有20%几率解冻；受到火属性招式攻击会解冻。
**免疫：** 冰属性宝可梦、熔岩铠甲特性。
**施加方式：** 冰冻光束/暴风雪的附加效果。
**移除方式：** 解冻药、利木果。
**VGC应用：** 较为罕见（因为附加效果概率低），但比睡眠更具破坏性，且没有固定的持续时间。

**非挥发性状态异常总结表**

| 状态异常 | 每回合HP损伤 | 能力变化             | 常见VGC施加者    |
| ---- | ------- | ---------------- | ----------- |
| 烧伤   | 1/16    | 攻击减半             | 炽焰咆哮虎 (鬼火)  |
| 中毒   | 1/8     | 无                | 晶光花(毒菱)     |
| 麻痹   | 无       | 速度减半 + 25%几率无法行动 | 闪电鸟 (电磁波)   |
| 睡眠   | 无       | 无法行动1-3回合        | 败露球菇 (蘑菇孢子) |
| 冰冻   | 无       | 无法行动，20%几率解冻     | 玛狃拉 (冰冻光束)  |

## III. 挥发性状态异常详解

挥发性状态异常种类多样，在VGC中用于即时干扰、支持或控场。

### A. 混乱

**效果：** 持续1-4回合，每回合有33%几率攻击自己（造成约40威力的伤害）。
**施加方式：** 奇异之光、虚张声势、暴风招式的附加效果。
**VGC应用：** 辅助控场，我行我素特性可以免疫。

### B. 畏缩

**效果：** 当回合无法行动（需要使用者先手攻击）。
**施加方式：** 击掌奇袭（拥有+3先制度，仅在出场第一回合有效）、岩崩。
**VGC应用：** 炽焰咆哮虎使用击掌奇袭进行开局控场。

### C. 挑衅

**效果：** 3回合内只能使用攻击类招式。
**施加方式：** 挑衅招式（恶作剧之心特性可使其获得先制度加成）。
**VGC应用：** 反制辅助类宝可梦，例如长毛巨魔挑衅败露球菇。

### D. 再来一次

**效果：** 锁定对手最后使用的招式，持续3回合。
**施加方式：** 再来一次招式。
**VGC应用：** 锁定对手的守住/顺风等招式。

### E. 注意力中心

**效果：** 对手的所有单体攻击招式强制以使用者为目标。
**施加方式：** 看我嘛/愤怒粉招式。
**VGC应用：** 保护核心队友进行战术强化，例如爱管侍使用看我嘛。

**其他常见挥发性状态异常：** 寄生种子（每回合吸取HP）、盐腌（每回合损失1/8 HP，钢/水属性宝可梦效果翻倍）、替身（提供伤害防护）。

**挥发性状态异常总结表**


| 状态异常  | 持续时间  | 效果        | VGC应用  | 移除方式 |
| ----- | ----- | --------- | ------ | ---- |
| 混乱    | 1-4回合 | 33%几率攻击自身 | 辅助干扰   | 切换出场 |
| 畏缩    | 1回合   | 无法行动      | 击掌奇袭开局 | 无    |
| 挑衅    | 3回合   | 禁止使用变化类招式 | 反制辅助   | 心灵香草 |
| 再来一次  | 3回合   | 锁定最后使用的招式 | 锁定无用招式 | 切换出场 |
| 注意力中心 | 直至切换  | 强制成为攻击目标  | 保护核心成员 | 切换出场 |
